Advanced WordPress security plugin with AI-powered threat detection, automatic IP blocking, and comprehensive protection system.

== Description ==

Morden Security is a comprehensive WordPress security solution that provides enterprise-level protection for your website. Built with modern security practices and powered by advanced threat detection algorithms.

**Key Features:**

* **Real-time Threat Detection** - AI-powered analysis of incoming requests
* **Automatic IP Blocking** - Dynamic blocking based on threat scoring
* **Advanced Bot Detection** - Distinguish between good and malicious bots
* **Web Application Firewall** - OWASP-compliant rule sets
* **Country-based Blocking** - Geographic access control
* **Behavioral Analysis** - Track and analyze visitor patterns
* **Performance Optimized** - Minimal impact on site speed
* **GitHub Updates** - Automatic updates from GitHub repository

**Security Modules:**

* Login Protection with rate limiting
* WordPress hardening and file protection
* Comment anti-spam with behavior tracking
* XML-RPC attack prevention
* Admin area protection
* Database security monitoring

**Advanced Features:**

* SQLite-based logging for performance
* RESTful API for integration
* Real-time dashboard monitoring
* Detailed threat analytics
* Custom rule creation
* Webhook notifications
* Multi-site network support

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/morden-security/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to 'Morden Security' in admin menu
4. Configure your security settings

== Requirements ==

* PHP 7.4 or higher
* WordPress 5.0 or higher
* SQLite3 PHP extension
* OpenSSL PHP extension
* 64MB RAM minimum (128MB recommended)

== Frequently Asked Questions ==

= Is this plugin compatible with other security plugins? =

Morden Security is designed to work alongside other security plugins, but we recommend testing in a staging environment first.

= Does it slow down my website? =

No, the plugin is optimized for performance with minimal overhead and efficient caching mechanisms.

= Can I whitelist specific IP addresses? =

Yes, you can create whitelist rules for trusted IP addresses or IP ranges.

== Privacy ==

This plugin collects and stores security-related data including:
* IP addresses of visitors (for security analysis)
* User agents and request details
* Geographic location data (country-level)
* Security event timestamps

All data is stored locally in your WordPress database and is not transmitted to external services unless explicitly configured.
